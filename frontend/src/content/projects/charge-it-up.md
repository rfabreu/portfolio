## Overview

Charge It Up is a web application that allows electric vehicle owners to quickly locate charging stations near their current location, covering all of North America.

## Problem

EV owners need a fast, reliable way to find nearby charging stations.

## Approach

Built a lightweight frontend application using vanilla HTML, CSS (Tailwind), and JavaScript with mapping API integration.

## Tech Stack

- **HTML/CSS/Tailwind** — Responsive, utility-first styling
- **JavaScript** — Maps API integration and geolocation
- **Third-party APIs** — Charging station data and mapping

## Outcome

A fast, user-friendly tool that helps EV owners find charging stations anywhere in North America.
